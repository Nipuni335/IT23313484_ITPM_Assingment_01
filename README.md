# IT23313484_ITPM_Assingment_01
Overview

This repository contains Playwright automation tests developed for ITPM Assignment 01.
The tests validate Singlish-to-Sinhala translation functionality of the SwiftTranslator web application.

Application Tested

https://www.swifttranslator.com/

Technologies

Node.js (LTS)

Playwright

JavaScript

Setup

Clone the public repository

git clone <YOUR_GITHUB_REPO_LINK>


Navigate to the project folder

cd ITPM_Assignment_01


Install dependencies

npm install

Run Tests
npx playwright test


View report:

npx playwright show-report
